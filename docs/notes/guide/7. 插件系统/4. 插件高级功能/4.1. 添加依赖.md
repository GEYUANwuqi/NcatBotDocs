## 依赖 Python 第三方库

部分复杂插件可能需要 **Python 第三方库**。

文件结构：
:::file-tree
- main.py
- plugins/
    - MyPlugin/
       - \_\_init\_\_.py
       - requirements.txt  <-- 新增此文件
       - plugin.py
:::

加载插件时会自动安装 `requirements.txt` 中列出的第三方库。

## 依赖其它插件（尚未完工）

使用 `BasePlugin.dependencies` 声明插件依赖, 格式见下.

```python
class MyPlugin(BasePlugin):
    name = "MyPlugin"
    version = "1.0.0"
    dependencies = {
        "OtherPlugin": ">=1.0.0",
        "LLM_API": ">=0.0.1"
    }
```
