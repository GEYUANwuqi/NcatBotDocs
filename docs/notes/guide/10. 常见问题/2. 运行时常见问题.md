### **QQ 提醒我使用了第三方插件**

无视它，如果被踢下线，重新登录即可。

要求安全验证时，按照它的要求做。

### **群聊消息无法接收，私聊信息仅接收notice**

在其他配置正常时出现此问题，大概是系统时间的原因。

对系统时间进行同步即可解决问题。

### **Please run this script in administrator mode .**

云电脑（云服务器）有安全设置，需要管理员权限才能运行，具体解决方式看具体运营商的规定。
