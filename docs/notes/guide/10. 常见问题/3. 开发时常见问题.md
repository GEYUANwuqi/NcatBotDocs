## 为什么我的插件没有加载成功

### 单文件插件

#### 必须使用 `__all__` 关键字指定需要导入的插件

```python
class MyPlugin(NcatBotPlugin):
    ...

__all__ = [MyPlugin] # 必须在 __all__ 中声明需要加载的插件。
```
